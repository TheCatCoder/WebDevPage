Course Reflection

I took this course because SLCC indicated that it could be used to fulfil part of my General Education requirements, and also because I felt that it would be fun and useful to know a little bit more about website design development. What I didn't anticipate is how useful it has been for my other programming tasks for me to learn more about web development. For example, I had been trying to learn how to manipulate a webpage using Powershell (a windows application that allows you to script certain computer tasks so that it can be done faster and without having to use the UI for the scripted application), knowing about how the DOM works and having manipulated my own webpages using Javascript made the whole task quite a bit more understandable. As I learn more about other programming languages in my free time, I can see that the knowledge and experience gained from this class will continue to be useful as I learn even more about programming and eventually start a career in a computer science field.
